# History Vibe Timeline

세계사의 주요 사건을 시대별로 탐색하고 복습할 수 있는 GitHub Pages용 정적 웹앱입니다.

## 주요 기능

- 세계사 주요 사건 연표 보기
- 지역별 필터링
- 키워드 검색
- 사건 카드 클릭 시 상세 설명 보기
- 랜덤 복습 퀴즈 기능

## 사용 방법

1. `index.html`, `style.css`, `script.js`, `README.md` 파일을 GitHub repository에 업로드합니다.
2. Repository Settings → Pages로 이동합니다.
3. Source를 `Deploy from a branch`, Branch를 `main`으로 설정합니다.
4. 생성된 GitHub Pages 링크를 과제로 제출합니다.

## 제작 의도

역사 공부에서 사건을 단순 암기하는 것이 아니라 시간의 흐름과 지역적 맥락 속에서 이해할 수 있도록 돕는 것을 목표로 했습니다.
